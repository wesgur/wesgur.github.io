# Johnny's Personal Website

This repository contains the source code of Johnny's personal website. 

A deployed version of this repository can be found [here](https://www.wesgur.com).

# Development

## Setup
1. Fill out `.env.example` and replace it to `.env`
2. Run `npm install` to install dependencies

## Local Environment
```sh
$ npm install
$ npm start
```

## Docker
```sh
$ docker-compose up
```
